Portfolio About

SurgeryVR (2018-2019)



Training VR simulator designed for invasive operations. Particular attention is paid to game mechanics, synchronized with real practitioners used in open operations.

Tasks: human tissues physics (cut/stretch), scenario system in inspector, mechanics, training module, lightning (static/dynamic), sound integration (adaptive/conditional), UI, localization.
Tech stack: Unity, C#, FMOD, VR (VRTK/OpenVR).

AIR (2020)



Informational MR application with locations info and person recognition.

Tasks: dynamic content, mechanics.
Tech stack: Unity, C#, Hololens, Mobile, MR (MRTK), AR (Vuforia/ARFoundation), Azure Spatial Anchor.

BioLab (2017)




Training desktop laboratory work to identify whether patients have systemic lupus erythematosus. This disease can be found using an enzyme-linked immunosorbent assay (ELISA, Elisa eng).

Tasks: node-based scenario system in unity editor (part), mechanics, chemical formulas math, UI, localization.
Tech stack: Unity, Unity Editor, C#.

Layerss (2019)




Entertaining AR application for Mutabor club.

Tasks: microphone equalizer, AR portal, UI/UX.
Tech stack: Unity, C#, Android, Shaderlab, AR (ARFoundation).

TatKitap (2019)




Entertaining AR application that allows you to ‘animate’ the characters of the coloring book "Peoples of Tatarstan Tales" by Tatar literature center.

Tasks: unfold 3D models, mechanics.
Tech stack: Unity, C#, Android, AR (Vuforia).

Workbooks (2019)



Educational mobile application for 3-4 years old children.

Tasks: UI/UX, database, user account, mechanics.
Tech stack: Unity, C#, Android, Shaderlab, After Effects.

Remote Assistant (2020)


Commercial AR application for remote assistance using video calls and spatial notes.

Tasks: location sync, spatial notes, UI.
Tech stack: Unity, C#, Android, Desktop, Shaderlab, AR (ARFoundation).

MuseumVR (2016)


Informational VR application about Bilarsk history museum.


Tasks: photogrammetry, mechanics, lightning (static/dynamic), UI.
Tech stack: Unity, C#, VR (NewtonVR).
